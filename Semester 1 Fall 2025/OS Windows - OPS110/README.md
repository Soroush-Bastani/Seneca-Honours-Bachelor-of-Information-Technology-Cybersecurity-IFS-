Subject Title
OS-Windows

Subject Description
In this course, the student will explore the fundamental concepts and operations of computer operating systems. O/S principles will be presented through a study of related terminology and practices. Topics will include operating system design and configuration, data storage and retrieval, network and security considerations, and management tools and services. Students will use the Windows operating system to demonstrate and apply operating system theory and principles.

Credit Status
1 credit in the IFS program.

Learning Outcomes
Upon successful completion of this subject the student will be able to:


    Install and configure Windows operating systems;
    Explain the purpose and concepts of operating systems using prescribed terminology;
    Describe operating system architecture using Windows as an example;
    Understand operating system processes and controls;
    Describe, support and maintain data and file storage and retrieval systems;
    Describe, support and ,modify services for internet access;
    Explain, utilize and refine management tools, services and utilities;
    Enable, update and support user account and group policies;
    Support and modify network services and security;
    Classify, distinguish and analyze configuration issues
